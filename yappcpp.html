one4all

The goals of the project is to provide an application interface for all major platforms. Using the most universal language C and C++ as the main language. The idea is to program once and deploy every where. This helps to reduce cost and time for any applicant while also maintain the functional integrity of the application it self. Some of the major platforms include.
  • Windows
  • Mac
  • Linux
  • Androind
  • Iphone
  • Single board computers
    • Raspberry PI
    • Orange PI
Some of the features includes
  • OpenGL OpenGLES
  • Streaming interface to media types
    • Jpeg, Png
    • mp3 and mp4
  • TTF fonts.
Streaming interfaces provides powerful ways to to receive and send any media item to file, Database, Network, memory mapped file, shared memory. It is both light weight and provides the most flexible option for any application.
